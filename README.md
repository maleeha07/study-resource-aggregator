# Contributing Guidelines

Follow the steps below to submit your solution for this task.

---

## 1. Fork the Repository

Click the **Fork** button at the top-right of this repository, or use the link below:

https://github.com/mulearngectcr/study-resource-aggregator

This will create a copy of the repository under your GitHub account.

---

## 2. Clone Your Fork

```bash
git clone https://github.com/<your-username>/study-resource-aggregator.git
cd study-resource-aggregator
```

---

## 3. Create a Folder with Your Name

Inside the repository, create a folder using your name and place your solution inside it.

You may also include:
- Notes about your implementation
- Sample outputs
- Screenshots (optional)

---

## 4. Project Requirements Checklist

Your solution must include:

- Python console application
- Input:
  - Topic of interest
  - Number of days for roadmap
- YouTube Data API integration
- Open Library API integration
- Aggregated data formatting
- LLM-based roadmap generation (mandatory)
- Secure API key handling using `.env`

---

## 5. Commit Your Changes

```bash
git add .
git commit -m "Add submission - Your Name"
```

---

## 6. Push to Your Fork

```bash
git push origin main
```

---

## 7. Create a Pull Request

1. Open your fork on GitHub  
2. Click **Contribute → Open Pull Request**  
3. Submit the pull request to the main repository  

---

## 📝 Pull Request Title Format

Submission - Your Name

Example:

Submission - John Doe

---

## ⚠️ Important Notes

- Create **only one folder with your name**.
- Do **not** modify or delete other submissions.
- Ensure your code runs before submitting the PR.
- Do **not commit `.env` files or API keys**.
- You may include bonus features like caching, ranking, or export options.

---

## 🚀 Bonus Challenge Ideas

- Add resource ranking by relevance
- Cache API responses to reduce repeated calls
- Export roadmap as JSON or TXT
- Add difficulty level detection (Beginner / Intermediate / Advanced)
- Improve LLM prompt quality for better roadmaps

---

Happy building 💻
